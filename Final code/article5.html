Article 5 – Principles of Data Processing

1. Lawfulness, Fairness and Transparency

Personal data must be processed lawfully, fairly and transparently, meaning users must clearly understand how their data is being used.

In this system, this is shown through the consent banner and privacy policy which explain how quiz data is stored.

2. Purpose Limitation

Data must be collected for specific, explicit and legitimate purposes and not used for anything else.

This system only uses data to track quiz results and does not reuse it for other purposes.

3. Data Minimisation

Data collected must be limited to what is necessary for the purpose.

This project does not collect personal information like names or emails, reducing privacy risks.

4. Accuracy

Personal data must be accurate and kept up to date, with incorrect data corrected or deleted.

The system ensures accuracy by recording quiz results directly from user input.

5. Storage Limitation

Data should only be stored for as long as necessary.

Users can delete their data at any time using the “Delete My Data” feature.

6. Integrity and Confidentiality

Data must be processed securely to prevent unauthorised access or loss.

This system stores data locally in the browser, meaning it is not shared externally.

7. Accountability

The organisation must be able to demonstrate compliance with all principles.

This project demonstrates accountability through consent, transparency, and user control over data.
