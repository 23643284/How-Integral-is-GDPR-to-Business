Article 7 – Conditions for Consent

1. Proof of Consent

Organisations must be able to demonstrate that a user has given consent.

This system records consent in localStorage when the user accepts the consent banner.

2. Clear and Accessible Consent

Consent must be clearly presented and easy to understand, using plain language.

The consent banner is simple and clearly explains data usage.

3. Right to Withdraw Consent

Users must be able to withdraw consent at any time, and it must be as easy as giving it.

This is implemented through the “Delete My Data” feature.

4. Freely Given Consent

Consent must be freely given and not forced or tied to unnecessary conditions.

Users are not forced to give consent and can decline it.
